# vrv-rbac
Assignment: Role-Based Access Control (RBAC)

## Installation
1. Clone the repository
```bash
git clone https://github
```

2. Setup Backend
```bash
cd vrv-rbac/backend
npm install
npm start
```

3. Setup Frontend
```bash
cd vrv-rbac/frontend
npm install
npm run dev
```
